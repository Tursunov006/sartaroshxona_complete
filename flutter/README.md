# Sartaroshxona Flutter Frontend

This Flutter app is a mobile/web frontend for the existing `backend` API.

## Features
- Browse services and barbers
- Create booking appointments
- View shops and Instagram links
- Submit and read reviews
- Search shops on a map

## Run
1. Open the `flutter` folder in your Flutter-enabled editor.
2. Run `flutter pub get`.
3. Start the app with `flutter run`.

## Backend URL
- Web: `http://localhost:5000/api`
- Android emulator: `http://10.0.2.2:5000/api`

If the backend runs on a different machine, update `lib/api_service.dart`.
